This is a collection of code in python to support the game found here: https://www.stockfighter.io

==========================================================================

Create a file named "keys.json" in the root directory, and add the following content there:

{
    "api_key" : <YOUR_API_KEY>
}

==============================================================================

Each level would be added in the format level<numberInWords>.py

For example, to play level one:

python levelone.py
